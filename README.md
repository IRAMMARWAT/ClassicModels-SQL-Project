# ClassicModels SQL Project

This project contains SQL queries for analyzing the ClassicModels database.

## Project Structure
- classicmodels_queries.sql - Main SQL queries
- data_analysis.sql - Advanced analysis queries
- eports.sql - Business reports

## Database Schema
The ClassicModels database contains tables for:
- Customers
- Orders
- OrderDetails
- Products
- Employees
- Offices
- Payments

## Usage
Run the SQL files in your MySQL database to analyze the ClassicModels data.
